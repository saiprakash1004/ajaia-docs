# AI Workflow Note

## AI Tools Used

- ChatGPT

---

## How AI Helped

AI was primarily used for:

- brainstorming architecture
- implementation guidance
- debugging
- reviewing code
- improving UI consistency
- identifying missing edge cases

---

## AI Output Rejected

Not every generated suggestion was accepted.

Some examples include:

- unnecessary complexity for TipTap configuration
- overly complicated sharing approaches
- features outside the required assessment scope

Those suggestions were simplified or rejected after manual review.

---

## Verification

Every AI-assisted change was manually verified by:

- running the application locally
- testing complete user workflows
- validating persistence
- checking deployment
- running automated tests

AI accelerated development, but all implementation decisions and final validation were completed manually.
